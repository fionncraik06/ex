About Me

Hi, I’m Fionn — a student in Dublin who loves building clean, responsive websites. I enjoy the overlap of design and code: turning simple ideas into fast, accessible pages with solid HTML, CSS and a growing dose of JavaScript. I’m currently focused on responsive layouts, semantic markup and small UI flourishes that make a site feel polished. Outside of coding, I’m into rugby and Star Wars; both keep me competitive and creative. This portfolio shares a selection of my projects, work experience and ways to contact me. Thanks for visiting!

Work Experience

My experience blends coursework, personal projects and part‑time roles.

  • Frontend tasks: built responsive pages, fixed layout bugs, and improved forms and navigation for accessibility.
  • Freelance sites: created small brochure sites for local groups, focusing on simple content structures and easy updates.
  • Collaboration: used Git to manage changes and documented features to help others pick up work quickly.

These projects taught me practical problem‑solving, clear naming, and how to structure HTML so CSS stays maintainable. I’m aiming for roles where I can keep improving my UI skills and learn modern tooling.
